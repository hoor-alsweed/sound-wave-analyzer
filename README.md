# 🎧 Sound Wave Analyzer

A Java-based application for analyzing and visualizing audio signals using fundamental signal processing techniques.

---

## 📌 Overview
This project was developed as part of a Java course in a collaborative team environment. It provides an interactive tool to load, process, and analyze audio signals using a modular design approach.

---

## 🚀 Features
- Amplitude visualization  
- Fast Fourier Transform (FFT) analysis  
- Power analysis  
- Signal filtering  
- Beat detection  
- Interactive graphical user interface  

---

## 🛠 Technologies Used
- Java  
- Signal Processing concepts  
- Object-Oriented Programming (OOP)  

---

## 📂 Project Structure
soundproject/
│
├── amplitude/
├── fft/
├── filter/
├── beat/
├── power/
├── AudioData.java
├── AudioAnalyzer.java
├── Main.java

---

## 🧩 Key Components
- **AudioData**: Handles loading and storing audio signal data  
- **AudioAnalyzer**: Coordinates different analysis operations on audio signals  
- **FFT Module**: Performs frequency-domain transformation of signals  
- Other modules handle filtering, power calculation, and beat detection  

---

## ▶️ How to Run
1. Open the project in IntelliJ IDEA or any Java IDE  
2. Compile and run the `Main` class  
3. Load an audio file (e.g., the provided sample)  
4. Explore different analysis features through the interface
